# 预备知识：张量及其运算

张量（tensor），本质上就是能够自定义维数、方便后续运算的数组。

## 形状和代码对应关系

```python
torch.tensor(3.14)                 # 0维，标量
torch.tensor(3,)                   # 1维，向量，一共3个数
torch.tensor([1, 2, 3])            # 1维，向量（直接赋值）
torch.tensor(2, 3)                 # 2维，2×3矩阵，一共6个数
torch.tensor([[1, 2], [3, 4]])     # 2维，2×2矩阵（直接赋值）
torch.randn(2, 3, 4)               # 3维张量，一共2×3×4=24个数
```

## 重要操作：自动求导

参与运算的都是张量，其中 `requires_grad=True` 标记的量作为要求导的量。

然后定义量与量之间的运算关系。

最后，对某量做 `.backward()` 求得该量对于需要求导的量的导数（偏导数），也即梯度。

### 举例

```python
x = torch.tensor(2.0)                        # 不求导
w = torch.tensor(3.0, requires_grad=True)     # 要求导
b = torch.tensor(1.0, requires_grad=True)     # 要求导
# 这里的三个量都是标量

y = x * w + b
loss = y ** 2       # 定义量与量之间的关系

loss.backward()     # 反向传播，求 dloss/dw 和 dloss/db
```

### 具体求导的形状

一般来说常用 $\dfrac{\partial s}{\partial \mathbf{X}}$（标量 $s$ 对张量 $\mathbf{X}$ 求导），得到的结果是和 $\mathbf{X}$ 形状相同的张量，该张量每个元素为 $s$ 对于 $\mathbf{X}$ 对应位置元素的偏导数。

### 举例

```python
X = torch.tensor([[1.0, 2.0], [3.0, 4.0]], requires_grad=True)

s = (X ** 2).sum()

s.backward()
```

这里

$s = 1^2 + 2^2 + 3^2 + 4^2$

所以：

$\frac{\partial s}{\partial X_{ij}} = 2X_{ij}$

于是梯度就是：

$X.\text{grad} = \begin{bmatrix} 2 & 4 \\ 6 & 8 \end{bmatrix}$
